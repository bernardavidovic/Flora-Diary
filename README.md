FLORA DIARY je stranica koja služi kao digitalni dnevnik za biljke s podsjetnikom za zalijevanje i općenitim podatcima o biljkama.
- otvara se desnim klikom na index.html s ekstenzijom Live Server u VSC-u




SADRŽAJ:
biljke.json - popis mojih biljaka

dodaj.html - stranica na kojoj se dodaju nove biljke

index.html - početna stranica

kontakt.html - opisna stranica

script.js - file za spremanje podataka i ostale interakcije

style.css - izgled stranice

vrste.html - stranica s vrstama biljaka

vrste.xml - popis vrsta biljaka
